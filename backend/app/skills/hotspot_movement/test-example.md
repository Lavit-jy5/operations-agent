# 热点异动测试样例

## 前端选择

- 题材类型：热点异动
- 表达风格：专业、克制、可发布
- 字数要求：350
- Skills：全部启用，至少启用「热点异动撰写」「催化筛选」「数据核验」「板块规则」

## 用户提示词

```text
【今日日期】：2026-07-13
【指定板块】：黄金

【今日信息】：
1. 用户从 Wind 终端记录：2026-07-13，AU9999 收盘价 890.00 元/克，日内涨幅 2.44%，来源 Wind。
2. 世界黄金协会在 2026-07-13 发布年中展望，认为下半年黄金表现将受地缘政治、利率环境和央行购金等因素影响，黄金仍可能充当全球宏观经济不确定性下的避险资产。
3. 用户记录：COMEX 黄金期货盘中站上 4180 美元/盎司，来源 CNBC 报价页，时间 2026-07-13。
4. 市场背景：美联储政策路径仍不明朗，美元指数阶段性走弱，中东地缘不确定性仍在。

请按热点速递格式输出一条主推方案和一个备选方案。
标题必须使用 AU9999 作为核心数据锚点。
机会解读控制在 110 字以内。
不要写投资建议，不要承诺收益。
```

## 期望结果检查

- 标题含「AU9999」或「黄金」。
- 标题包含 2.44% 或 890.00 元/克等用户提供的数据。
- 机会解读不超过 110 字。
- 正文包含主推方案、备选方案、撰写说明、自查清单。
- 风险提示中不出现买入建议、确定性上涨、收益承诺。
- 若模型使用 COMEX 4180 美元/盎司，必须能在引用或撰写说明里标明 CNBC/2026-07-13。
